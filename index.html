Major Projects

K-Restricted Boltzmann Machines (K-RBMs) PPT

We describe a novel approach to non-linear subspace clustering that employs Restricted Boltzmann Machines. Our model (a) makes no assumptions about the nature of subspaces sought, and (b) it solves the ``coupled" problem of learning the subspaces and projections simultaneously. We apply this clustering framework to understanding general data. Our hypothesis here is: in general, data is embedded in multiple non-linear subspaces and and within each manifold there may be further clusters. We discuss an overall solution to understanding data that first finds multiple non-linear sub-spaces within the data using K-RBMs and then further clusters the data within each sub-space linearly. We describe a BoW pipeline that uses two-level clustering: non-linear clustering first and linearly clustering each manifold further. We further apply our K-RBMs clustering framework for learning dense local K-RBM features from raw image patches.

Hierarchical Bag of Words using Naive Bayes Clustering PPT

We devise a novel Naive Bayes clustering framework for symbolic data. We employ our clustering model to cluster symbolic vectors of BoW visual words in a neighbourhood, to learn higher level features in a deep learning fashion. In doing so, we incorporate the traditional BoW model with means to exploit the spatial context in images.

Partial Least Squares Kernel for computing similarities between Videos PPT

We describe a novel strategy of computing similarity between two video sequences that uses the Partial Least Squares regression. Our approach uses no parameter tuning. We demonstrate the use of this similarity kernel to solve the tasks of hand gesture recognition on the Cambridge dataset and human activity classification on the UCF sports dataset.

Sparse Discriminative Fisher Kernels for Visual Classification PPT

Most practical solutions require us to understand and resolve the tradeoff between performance and resources. Fisher vectors demonstrate this concept well. The representative power of Fisher vectors gives excellent classification performance but comes at a storage and computational cost. We describe strategies to introduce sparseness and discriminative information in Fisher vector encodings. We discuss implementation tricks that enable us to reap the advantages of huge vocabularies using representations whose explicit sizes are much smaller.

Which visual words are important in a discriminative task?

Here we seek simple, intuitive solutions to two fundamental questions, (a) which visual words are important, and (b) which codebooks are important. This knowledge can be used to enhance a vocabulary by rejecting visual words with little discriminative power, or select an optimal set of vocabularies for a task given a large pool of vocabularies. It may be pointed out that while this form of subset selection is an NP hard problem per se due to the combinatorial explosion, we describe a greedy approach which is linear in the number of visual words. We discuss a strategy to mix and match visual words drawn from different codebooks.

Other Projects

TREC Video Retrieval

We participated in TRECVID 2010, organized by NIST. Our pipeline had dense SIFT features, Spatial Pyramids based Bag of Words representation with SVM classifiers for scene category retrieval and a deformable part based models for object detection in videos. We combined evidence of presence of a class of object classification, and detection approaches using the Borda Count method.

Detecting, Tracking and Recognizing Humans in Hollywood Movies

We used a deformable parts based model to detect human upper bodies in video frames. We combined detections into tracks, and actor recognition was based on colour features.

Similar Image Search

We extracted simple and efficient colour features from images, encoded these into text and used the open source text search engine Lucene to index a database of images and retrieve images similar to a query image.

Clustering NEWS Videos based on Visual Content

We used image processing to segment huge NEWS video volumes into relevant stories, reject advertisements and other clutter using SVM classifiers. We use colour features to hierarchically cluster these video-stories.
